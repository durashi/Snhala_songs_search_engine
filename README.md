# Snhala_songs_search_engine

Corpus including 2270 Sinhala songs each with 10 attributes(meta data).

##Meta data
track_id - track identifier
track_name_en - track name in Sinhala+English
track_name_si - track name in Sinhala unicode
track_rating - ratings for the song track
album_name_en - album name in Sinhala+English
album_name_si - album name in Sinhala unicode
artist_name_en - artist name in Sinhala + English
artist_name_si - artist name in Sinhala unicode
artist_rating - ratings of the artist
lyrics - Sinhala lyrics of the song in Sinhala unicode
